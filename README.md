# 얼루가게 fe 저장소입니다.

nextjs를 사용한 프로젝트입니다.
public, modules 디렉토리는 서브모듈로 관리됩니다.
서브모듈의 수정은 함부로 진행하지 마시고 수정사항을 알려주세요.

## 시작방법

```bash
git clone (저장소 주소) --recursive
```

으로 저장소를 클론 받아주세요.

## 개발환경

```bash
yarn install
yarn dev
```

으로 개발환경을 띄워주세요.

### 서브모듈

서브모듈을 최신상태로 업데이트 하기위해

```bash
git submodule update --recursive
```

으로 업데이트르 진행해주세요.
